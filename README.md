# Task Manager API (Flask)

A simple Flask REST API to manage tasks (CRUD operations) using SQLite.

## Features
- Create a new task
- View all tasks
- Update task status
- Delete tasks
- SQLite database for storage

## Tech Stack
- Python
- Flask
- SQLite

```bash
python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
